# ADDIS-KETEMA--EDUTRACK-GRADING-
This is Group project for Student Grading System 
**Overview**
Addis Ketema EduTrack is a comprehensive, web-based academic management system engineered for the Addis Ketema preparatory school environment. The platform is designed to streamline administrative, faculty, and student workflows by providing centralized access to grading, notifications, and student progress tracking.

**🌐 Access the Platform**
You can explore the live version of the Addis Ketema EduTrack system here: [https://addis-ketema-edutrack-243499697994.europe-west2.run.app](https://addis-ketema-edutrack-243499697994.europe-west2.run.app)
*We invite you to navigate through the dashboards to experience the integrated administrative and academic tracking features.*

**⚙️ Technical Architecture**
The application is built upon a modern, high-performance stack designed for scalability and developer efficiency:

 **Frontend Framework:** Built with React and TypeScript, enabling a modular, type-safe architecture that ensures robust data handling and a seamless user interface.
 **Design & Styling:** Utilizes Tailwind CSS to implement a responsive, high-fidelity, and modern aesthetic, ensuring the platform is accessible across all devices.
 **Database & Backend Services:** Leverages Supabase as a comprehensive Backend-as-a-Service (BaaS), handling secure authentication, real-time database operations, and API management without the need for traditional server-side infrastructure.
 **Development Environment:** Powered by Vite, providing an optimized and lightning-fast build pipeline for modern web development.

**  Key Features**
**Administrative Dashboard:** Centralized control for managing student records, faculty access, and institutional data.
 **Faculty Dashboard:** Intuitive interface for inputting, updating, and tracking student grades across various subjects.
 **Student Dashboard:** Personalized portal for students to view academic progress, performance metrics, and school notifications.
 **Secure Authentication:** Robust user login and role-based access control to ensure data privacy and security.
 **Real-time Updates:** Dynamic data handling for immediate reflection of grade inputs and system notifications.

** Key Objectives**
The system serves to digitize academic documentation, replacing traditional manual processes with an efficient, data-driven platform. By integrating specialized modules for different school stakeholders—Administrators, Faculty, and Students—Addis Ketema EduTrack improves institutional transparency and facilitates better academic outcomes.
